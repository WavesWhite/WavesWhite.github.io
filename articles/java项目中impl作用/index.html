java项目中impl作用

发表于2022-10-24|更新于2022-10-25|杂七杂八
|字数总计:106|阅读时长:1分钟|阅读量:

在Java项目开发中，通常会将后台分成几层，常见的是三层MVC：

  • Model（模型层）
  • View（视图层）
  • Controller（控制层）

而impl通常处于Controller层的service下，是用于存放service接口的实现类的

impl的全称是implement，表示实现的意思，在Java语法中是实现接口的意思，所以impl在Java项目中是用于实现接口的
